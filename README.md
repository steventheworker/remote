# remote [administration tool]
## Dependencies
###  nodejs
  - sockjs
  - python-shell
  - node-static
###  python
- mss
- Pillow
- pynput
- pywin32
## install
run in console
```
npm install
```
```
pip3 install mss
pip3 install Pillow
pip3 install pynput
pip3 install pywin32
```
## start
```
npm start
```
